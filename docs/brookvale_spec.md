## 2. 🌍 세계관 (Worldview)

### 2.1 핵심 컨셉
**"당신의 일상이 흐르는 마을 (Where Your Daily Life Flows)"**

- **지명**: Brook(시냇물) + Vale(골짜기) = **Brookvale**
- **비주얼**: 지브리풍 수채화 + 픽셀 아트 혼합
- **사용자 역할**: 마을의 **관리자(Caretaker)** + **주민(Villager)**

### 2.2 시냇물 (The Stream) - 핵심 요소!
시냇물은 단순 장식이 아닌 **세계관의 심장**입니다.

| 의미 | 설명 |
|------|------|
| **시간의 흐름** | 상류(아침) → 하류(밤) = 하루의 여정 |
| **성장의 척도** | 활동량에 따라 물줄기가 커짐 |
| **감정의 연결** | 물수제비로 마음을 전함 |

**성장 단계**: 이슬(Dewdrop) → 옹달샘 → 시냇물 → 강물 → **바다(Ocean)**

### 2.3 하루의 루틴
```
아침의 계획(숲) → 낮의 실천(마을) → 저녁의 회고(호수) → 밤의 꿈(언덕)
```
