## 💰 수익 모델 (3-Tier Strategy)

### 🆓 Tier 1: Free (무료 - 시드 확보)
> 목표: 유저 획득 & 습관 형성

**포함 항목:**
- 기본 17개 앱 전부 사용 가능
- 일일 도토리 5개 제한 (현재 무제한 → 제한)
- 기본 캐릭터 1개
- 주간 통계 기본 보기
- 3D 월드 자유 탐험

**제한 사항:**
- 히스토리 최대 7일만 보기
- 차트/분석 기본형만
- 사운드스케이프 2개만
- 루틴 커스텀 불가

---

### ⭐ Tier 2: Valley Pass (밸리 패스) — $4.99/월 or $39.99/년
> 목표: 핵심 수익원 (80% 매출 비중 목표)

**추가 혜택:**
| 기능 | 설명 |
|---|---|
| 🌰 **도토리 무제한** | 일일 수집 제한 해제 |
| 📊 **전체 통계 해금** | 30일/90일/전체 히스토리 + 분석 차트 |
| 🎵 **전체 사운드스케이프** | 6종 모두 해금 (현재 2종 무료) |
| 🎨 **테마 & 스킨** | 다크모드, 시즌 테마, 커스텀 컬러 |
| 🏠 **캐릭터 확장** | 추가 아바타 5종 + 의상 |
| 🌲 **성장 시스템 확장** | 참나무 → 벚꽃나무, 크리스탈 트리 등 |
| ✨ **프리미엄 루틴** | 커스텀 루틴 생성 + AI 추천 |
| 🏆 **독점 퀘스트** | 주간 스페셜 퀘스트 + 보너스 도토리 |
| 📱 **오프라인 모드** | PWA 완전 오프라인 지원 |
| 🔔 **리마인더** | 푸시 알림 (루틴, 명상, 취침) |

**가격 산정 근거:**
- Calm: $14.99/월 → Brookvale는 명상 + 생산성 + 게이미피케이션 올인원
- Habitica: $4.99/월 → 비슷한 포지셔닝
- 타겟: **$4.99/월** (저항 없는 "커피 한 잔" 가격)
- 연간: **$39.99** (33% 할인) → 연간 전환 유도
